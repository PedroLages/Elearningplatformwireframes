### What Works Well

1. **Event propagation for `?` key**: Capture-phase handler on `document` prevents Layout-level `?` from firing.

3. **Clean component extraction**: `VideoShortcutsOverlay` is stateless, well-typed, with `data-column` test attributes.

### Findings

#### Blockers

- **B1 — `data-testid="video-player"` renamed to `video-player-container`** (`VideoPlayer.tsx:621`): Three existing E2E tests reference `getByTestId('video-player')`. Fix: Restore `data-testid="video-player"` or update the 3 test files.
- **B2 — Shift+Arrow shortcut documented but not implemented** (`VideoShortcutsOverlay.tsx:30`, `VideoPlayer.tsx:458-528`): Overlay lists `Shift + ←→` as "Seek ±10s" but no handler exists. Fix: Remove the entry or implement the handler.

#### High Priority

- **H1 — Overlay missing dialog ARIA semantics** (`VideoShortcutsOverlay.tsx:62-64`): No `role="dialog"`, focus management, or focus trap.
- **H2 — Close button 36x36px** (`VideoShortcutsOverlay.tsx:70`): Below 44x44px minimum. Use `size-11`.
- **H3 — Missing E2E tests**: No test clicks skip buttons (different code path than J/L). No test for PiP exit flow.

#### Medium

- **M1 — `announceTimeoutRef` not cleared on unmount** (`VideoPlayer.tsx:293-297`).
- **M2 — Pre-existing `exhaustive-deps` violations** in `handleEnded`, `togglePlayPause`, `seekToTime`.
- **M3 — Dead `transition-opacity duration-200` on overlay** (component returns `null` when closed).

#### Nits

- **N1**: `min-w-[24px]` → `min-w-6` (Tailwind v4 supports it).
- **N2**: `typeof document !== 'undefined'` unnecessary in Vite SPA.
- **N3**: `?` key dispatch consistency is good in final version.
